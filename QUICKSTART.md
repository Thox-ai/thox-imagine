## 📝 Basic Usage

### Option 1: Command Line (Simplest)

```bash
# Make CLI tool executable (first time only)
chmod +x thox-imagine-cli.sh

# Generate an image
./thox-imagine-cli.sh "A beautiful sunset over mountains"

# Interactive mode (guided)
./thox-imagine-cli.sh --interactive

# View examples
./thox-imagine-cli.sh --examples

# See all options
./thox-imagine-cli.sh --help
```

### Option 2: Direct Ollama Command

```bash
ollama run Thox-ai/thox-imagine "Your prompt here"
```

### Option 3: Python API

```bash
# Make Python script executable (first time only)
chmod +x thox_imagine_api.py

# Generate an image
python3 thox_imagine_api.py "A serene Japanese garden"

# Save to specific file
python3 thox_imagine_api.py "Cyberpunk city" --output mycity.png

# Show model info
python3 thox_imagine_api.py --info

# List generated images
python3 thox_imagine_api.py --list
```

### Option 4: Demo Interface

```bash
# Make demo executable (first time only)
chmod +x thox-imagine-demo.sh

# Run interactive demo
./thox-imagine-demo.sh
```

## 🔧 Integration Examples

### Python Integration

```python
from thox_imagine_api import ThoxImagineAPI

# Initialize API
api = ThoxImagineAPI()

# Generate single image
image_path = api.generate_and_save(
    "A majestic waterfall in ancient forest"
)

# Batch generation
prompts = [
    "A sunset over the ocean",
    "A futuristic city",
    "An ancient temple"
]
paths = api.batch_generate(prompts)
```

### Node.js/TypeScript Integration

```typescript
import { ThoxImagineClient } from './thox-imagine-node-example';

const client = new ThoxImagineClient();

// Generate image
const result = await client.generate(
    'A serene mountain landscape'
);

console.log('Image saved to:', result.imagePath);
```

### cURL / REST API

```bash
curl -X POST http://localhost:11434/api/generate \
  -H "Content-Type: application/json" \
  -d '{
    "model": "Thox-ai/thox-imagine",
    "prompt": "A beautiful landscape",
    "stream": false
  }'
```

## 🆘 Troubleshooting

### Model Not Found
```bash
# Recreate the model
ollama create Thox-ai/thox-imagine -f Modelfile.thox-imagine
```

### Slow Generation
- Ensure GPU acceleration is enabled
- Close other resource-intensive applications
- Consider using smaller image resolutions

### Out of Memory
- Reduce batch size
- Generate one image at a time
- Increase system RAM if possible
