## 🛠️ Customization

### Modify Model Behavior

1. Edit the Modelfile:
   ```bash
   nano Modelfile.thox-imagine
   ```

2. Modify parameters:
   - `PARAMETER temperature 0.8` - Adjust creativity (0.0-1.0)
   - `PARAMETER top_p 0.95` - Adjust sampling (0.0-1.0)
   - `SYSTEM` block - Change generation instructions
   - `TEMPLATE` block - Modify prompt template

3. Recreate the model:
   ```bash
   ollama create Thox-ai/thox-imagine -f Modelfile.thox-imagine
   ```

### Modify Tools

- **CLI tool**: Edit `thox-imagine-cli.sh`
- **Python API**: Edit `thox_imagine_api.py`
- **Demo script**: Edit `thox-imagine-demo.sh`
- **Node.js**: Edit `thox-imagine-node-example.ts`
